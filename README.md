# 📊 Sales Data Analysis Using Core Python

## 📌 Project Overview

This project demonstrates sales data analysis using **Core Python** without relying on external data analysis libraries such as Pandas or NumPy.

The project reads sales data from a CSV file, performs data cleaning, analyzes key business metrics, answers common sales-related questions, and exports filtered data into a new CSV file.

The primary objective of this project is to strengthen Python fundamentals, including file handling, lists, dictionaries, loops, conditional statements, and date manipulation while solving real-world business problems.

---

## 🎯 Project Objectives

The objective of this project is to analyze sales data using Core Python and extract meaningful business insights from a sales dataset.

The project focuses on:

- Reading and processing CSV data using file handling.
- Cleaning and preparing raw sales data.
- Calculating important sales metrics.
- Performing customer, product, category, and date-wise analysis.
- Exporting filtered sales records into a separate CSV file.
- Strengthening Core Python programming and problem-solving skills.

---

## 📂 Dataset

The project uses a sales dataset stored in a CSV file containing **1,000 sales records**.

### Dataset Columns

| Column | Description |
|---------|-------------|
| OrderID | Unique order identifier |
| Date | Date on which the order was placed |
| Customer | Customer name |
| Product | Product purchased |
| Category | Product category |
| Quantity | Number of units sold |
| Price | Price per unit |

Input Dataset:

```text
sales_data.csv
```

Generated Output File:

```text
electronics_orders.csv
```

---

## 🧹 Data Cleaning

Before performing the analysis, the dataset is cleaned by:

- Removing the header row.
- Splitting each row into individual fields.
- Converting **Quantity** into an integer.
- Converting **Price** into an integer.
- Storing the cleaned records into a new list for further analysis.

---

## 📊 Analysis Performed

The project performs the following business analyses:

1. Calculate the total revenue from all sales.
2. Count the total number of orders.
3. Calculate the total quantity of items sold.
4. Calculate revenue generated by each category.
5. Calculate revenue generated by each product.
6. Identify the product with the highest quantity sold.
7. Identify the customer who spent the most money.
8. Calculate total revenue generated for each date.
9. Calculate revenue between two user-specified dates.
10. Export all orders belonging to the **Electronics** category into a separate CSV file.

## ▶️ How to Run

1. Clone this repository.

```bash
git clone https://github.com/your-username/Sales-Data-Analysis-Using-Core-Python.git
```

2. Open the project folder.

3. Launch **Jupyter Notebook** or open the notebook in **VS Code**.

4. Open:

```text
SalesAnalyzer.ipynb
```

5. Run all cells sequentially.

6. For Question 9, enter the start date and end date in the following format:

```text
YYYY-MM-DD
```

7. Running Question 10 will generate:

```text
electronics_orders.csv
```

containing all orders under the **Electronics** category.
