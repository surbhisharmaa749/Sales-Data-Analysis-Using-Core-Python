# 📊 Sales Data Analysis Using Core Python

## 📌 Project Overview

This project demonstrates sales data analysis using **Core Python** without relying on external data analysis libraries such as Pandas or NumPy.

The project reads sales data from a CSV file, performs data cleaning, analyzes business metrics, answers common sales-related questions, and exports filtered data into a new CSV file.

The primary objective of this project is to strengthen Python fundamentals, including file handling, lists, dictionaries, loops, conditional statements, and date manipulation while solving real-world business problems.

---

## 🎯 Project Objectives

The objective of this project is to analyze sales data using Core Python and extract meaningful business insights without using external data analysis libraries.

The project focuses on:

- Reading and processing CSV data using file handling.
- Cleaning and preparing raw sales data for analysis.
- Calculating key sales metrics such as revenue, orders, and quantity sold.
- Performing customer, product, category, and date-wise analysis.
- Exporting filtered sales records into a new CSV file.
- Strengthening Core Python programming and problem-solving skills through a real-world dataset.
