# 📊 Sales Data Analysis Using Core Python

## 📌 Project Overview

This project demonstrates sales data analysis using **Core Python** without relying on external data analysis libraries such as Pandas or NumPy.

The project reads sales data from a CSV file, performs data cleaning, analyzes business metrics, answers common sales-related questions, and exports filtered data into a new CSV file.

The primary objective of this project is to strengthen Python fundamentals, including file handling, lists, dictionaries, loops, conditional statements, and date manipulation while solving real-world business problems.
